# 🚀 ET Compass – AI Concierge for Economic Times

## 📌 Overview

ET Compass is an AI-powered concierge that helps users navigate the vast Economic Times ecosystem through a simple conversational interface.

Instead of browsing multiple sections manually, users answer a few questions and receive **personalized recommendations** across ET Prime, Markets, Finance, Masterclasses, and more.

---

## 🎯 Problem Statement

Users often explore only a small portion of the Economic Times platform due to:

* Information overload
* Lack of personalization
* Difficulty discovering relevant content

ET Compass solves this by acting as a **smart guide** that understands user intent and recommends the right ET resources instantly.

---

## 💡 Solution

We built a **conversational AI-driven web app** that:

1. Understands the user through a quick chat
2. Builds a simple financial profile
3. Maps the user to relevant ET ecosystem sections
4. Displays curated, actionable recommendations

---

## ⚙️ How It Works

1. User clicks **Start**
2. Chat collects user preferences:

   * Role (student, working, investor)
   * Goal (learn, grow, track)
   * Risk level
3. System processes responses
4. Dashboard displays **filtered ET content**
5. User clicks cards → redirected to ET platform

---

## 🧠 Personalization Logic

Basic rule-based filtering:

* **Learn Investing →**

  * ET Prime
  * Masterclass

* **Grow Wealth →**

  * Markets
  * Finance

* **Track Markets →**

  * Live Markets
  * IPO section
